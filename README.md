# HexSchool React 2025 課程作業

## w2 功能介紹

1. 串接登入和後台產品列表 api
2. 成功登入後將 token 存在 cookie 裡
3. 後台所有需要 token 驗證都會在載入時驗證 token，判斷直接登入或返回登入頁
4. 點擊產品列表的查看，可以查看訂單詳細頁
5. 所有頁面油 router 分頁

## w3 功能介紹

1. 增加新增，編輯，刪除列表的功能
2. 串接新增，編輯，刪除列表的api
3. 啟用欄位可依資料結果切換顏色

## w4 功能介紹

1. 將 modal 內容元件化
2. 新增分頁 pagination 功能
3. 新增圖片上傳功能
4. 新增欄位

## w5 功能介紹

1. 建立前台路由表
2. 建立前台頁面 - 首頁/產品頁/產品詳細頁/購物車頁/404頁
3. 可查看產品頁和詳細頁，並加入購物車
4. 可查看購物車頁面，並進行刪除和更改數量

## 使用技能

1. bootStrap, axios
2. React useState, useEffect. useRef
3. React router
